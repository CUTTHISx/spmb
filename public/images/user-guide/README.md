Folder ini berisi screenshot untuk dokumentasi User Guide PPDB Online.

## Struktur File

Simpan screenshot dengan naming convention:
```
{nomor}_{nama_fitur}.png

Contoh:
- 01_homepage.png
- 02_login_page.png
- 48_verifikasi_bayar.png
```

## Cara Menggunakan

1. Ambil screenshot sesuai panduan di `SCREENSHOT_GUIDE.md`
2. Simpan di folder ini dengan nama yang sesuai
3. Update `USER_GUIDE.md` dengan menambahkan referensi gambar

## Format Gambar

- **Format**: PNG
- **Max Size**: 500KB
- **Resolution**: 1920x1080 atau 1366x768

## Placeholder

Jika screenshot belum tersedia, gunakan placeholder:
```
[Screenshot: Nama Fitur]
(Gambar akan ditambahkan)
```
